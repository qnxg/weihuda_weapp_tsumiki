# front_2026 设计方案

## 1. 项目结构

### 1.1 页面

#### 1.1.1 主包 `src/pages/`

1. login - 登录页
2. index - 首页
3. toolkit - 工具箱
4. table - 课表
5. profile - 我的
6. feedback - 反馈
7. faq - 常见问题
8. disclaimers - 免责声明
9. message - 消息盒子

#### 1.1.2 Tools 包 `src/tools/pages/`

1. grade/grade - 成绩查询
2. grade/ranking - 成绩排名
3. grade/experiment-grade - 实验成绩
4. grade/physical-grade - 体测成绩
5. exam/experiment-arrange - 实验安排
6. exam/exam-arrange - 考试安排
7. exam/physical-appoint -- 体测预约
8. exam/custom-exam - 自定义考试
9. campus/calender - 校历查看
10. campus/physical-standard - 体测标准
11. campus/empty-room - 空教室查询
12. campus/card-bill - 校园卡账单
13. campus/netflow-bill - 校园网账单
14. campus/netflow-detail - 流量详情
15. campus/electricity - 电费查询

#### 1.1.3 Setting 包 `src/setting/pages/`

1. index-card - 首页卡片
2. lab-bind - 大物平台绑定
3. class-table - 课表设置
